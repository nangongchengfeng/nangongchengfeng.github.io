1、编写

自定义mycentos目的使我们自己的镜像具备如下：

  1.          登陆后的默认路径
  2.          vim编辑器
  3.          查看网络配置ifconfig支持

 #基于本地的centos
FROM centos   
 #作者、邮件
MAINTAINER cf<1794748404@qq.com>
 #来设置环境变量
ENV MYPATH /uer/local 
#登录进去的路径
WORKDIR $MYPATH
##安装下面的软件
RUN yum -y install vim
RUN yum -y install net-tools
#暴露80端口
EXPOSE 80
#打印信息
CMD echo $MYPATH
CMD echo "success-----------------ok"
#使用bash
CMD /bin/bash

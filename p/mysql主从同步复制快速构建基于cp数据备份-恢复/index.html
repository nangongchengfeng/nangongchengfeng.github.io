环境：

主库：    mysql

从库：        mysql

思路

（1）主库A数据全，数据库B空的

（2）锁表数据库A，记录pos号

（3）复制数据库A目录数据（进行cp备份）然后 解锁表

（4）把数据库A备份的数据进行scp 数据库B

（5）把数据库A的目录，进行覆盖数据库B的目录。（数据覆盖，相当于备份恢复）

（6）进行主共同步

**注意：日志格式 **statement  row mixed   建议使用 mixed

3

/etc/init.d/iptables restart
